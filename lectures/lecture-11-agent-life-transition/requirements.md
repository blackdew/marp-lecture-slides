# 강의 요구사항 정의서

## 기본 정보

| 항목 | 내용 |
|------|------|
| 강의명 | Agent 활용 생활/업무 전환 (가제) |
| 대상 | 동명대학교 대학생/대학원생 및 교직원 |
| 날짜 | 2026년 2월 말 |
| 시간 | 2시간 |
| 인원 | 20~50명 (중규모) |
| 장소 | 미정 (개인 노트북 지참) |

## 수강 대상 특성

- **구성**: 대학생, 대학원생, 교직원이 혼재된 그룹
- **AI 경험 수준**: 초급~중급 다양 (ChatGPT 기본 사용부터 업무 활용까지)
- **핵심 니즈**: 각자의 업무와 학업에 AI Agent를 실제로 활용하는 방법

## 강의 목표

1. AI Agent의 개념과 기존 챗봇과의 차이를 이해한다
2. 각자의 역할(학생/대학원생/교직원)에 맞는 Agent 활용 시나리오를 파악한다
3. 실제 도구를 사용하여 Agent 기반 작업을 직접 경험한다
4. 강의 후 스스로 활용을 확장할 수 있는 기초 역량을 갖춘다

## 고려사항

### 수강생 다양성

- 전공/직무가 다양하므로 **범용적인 활용 사례** 중심 구성 필요
- 학생: 학업(리서치, 보고서, 발표자료 등)
- 대학원생: 연구(논문 탐색, 데이터 분석, 문헌 정리 등)
- 교직원: 업무(문서 작성, 데이터 정리, 회의록 처리 등)

### 실습 환경

- 개인 노트북 지참 → 실습 가능
- AI 경험 수준 격차가 클 수 있으므로 **단계적 실습 설계** 필요
- 무료/저비용 도구 우선 활용 (유료 구독 전제 최소화)

### 시간 배분 (2시간)

- 이론/개념: 충분히 짧게 (전체의 30% 이내)
- 시연/실습: 중심 (전체의 50% 이상)
- Q&A/정리: 마무리 (전체의 20%)

## 성공 기준

- 수강생이 "나의 업무/학업에 이걸 어떻게 쓸 수 있겠다"는 감을 잡는 것
- 최소 1개 이상의 Agent 활용 작업을 직접 수행해보는 것
- 강의 후 혼자서도 시도할 수 있는 시작점을 제공하는 것
