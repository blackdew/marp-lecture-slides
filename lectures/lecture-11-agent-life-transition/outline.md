# Just Plan It!! - AI Agent로 시작하는 일과 삶의 전환

**강의 제목**: Just Plan It!! - AI Agent로 시작하는 일과 삶의 전환
**대상**: 동명대학교 대학생
**시간**: 2시간 (120분)
**형식**: 이론 30% + 실습 40% + 체험/토론 30%

---

## 강의 핵심 목표

**한 줄 요약**: AI Agent를 직접 체험하고, 대학 생활과 미래 커리어에서 AI와 협업하는 관점을 장착한다

**3가지 핵심 역량**:

1. **AI Agent 체험**: 챗봇을 넘어선 AI Agent의 실체를 직접 경험
2. **AI 시대 관점 전환**: 선택, 계획, 문서화 - 3가지 키워드로 새로운 시각 장착
3. **일과 삶의 적용**: 학업, 취업 준비, 사이드 프로젝트에 AI Agent 활용법 체득

---

## 대상 분석: 동명대학교 대학생

**특성**:
- 디지털 네이티브 세대 (ChatGPT, Copilot 등 AI 경험 있음)
- 학업 효율화에 관심 (과제, 리포트, 발표 자료)
- 취업/진로에 대한 고민 (AI 시대 역량)
- 사이드 프로젝트/창업에 관심 있는 학생 존재

**기대하는 것**:
- "ChatGPT 말고 뭐가 다른 거야?"에 대한 답
- 실제로 써먹을 수 있는 실용적 경험
- AI 시대에 대학생으로서 어떤 역량을 키워야 하는지

## Part별 상세 구성

### Part 1: 오프닝 - AI Agent, 왜 지금인가? (15분)

**목표**: 흥미 유발 + AI Agent 개념 이해

**슬라이드 구성** (4장):
1. 타이틀 슬라이드 (강의 제목, QR코드)
2. 오늘의 여정 (전체 타임라인)
3. 챗봇 vs Agent 비교 (표 + 비유)
4. Agent 구조: LLM + Memory + Planning + Tools

**핵심 메시지**: "Agent = 도구를 쓸 줄 아는 AI 동료"
**오프닝 훅**: "ChatGPT에게 '파일 만들어줘'라고 해본 적 있나요?"

---

### Part 2: 첫 체험 - Agent와 함께하기 (20분)

**목표**: 성공 경험을 통한 "와, 이게 되네!" 체감

**슬라이드 구성** (5장):
1. CLI 도구 비교 (Claude Code / Gemini / Codex)
2. 설치 및 첫 실행
3. 실습 2-1: 자기소개 페이지 만들기 (HTML)
4. 실습 2-2: Plan 모드 체험
5. 중간 회고: "어디에 써먹을 수 있을까?"

**실습 2-1: 자기소개 페이지 만들기**
- "나를 소개하는 HTML 페이지를 만들어줘"
- 이름, 전공, 관심사, 좋아하는 것
- 브라우저에서 열어보기

**실습 2-2: Plan 모드 체험**
- Shift+Tab 두 번 → Plan 모드 진입
- "자기소개 페이지에 다크모드 토글 추가해줘"
- 계획 확인 → 승인 → 실행

---

### Part 3: AI 시대 3가지 키워드 (25분)

**목표**: 체험을 개념으로 정리 + 관점 전환

**슬라이드 구성** (7장):
1. Part 3 타이틀
2. 키워드 1: 선택 (Choice) - "AI가 해도 결정은 당신의 몫"
3. LLM 작동 원리 (확률 기반 예측)
4. 키워드 2: 계획 (Planning) - "좋은 프롬프트 = 좋은 계획"
5. 좋은 프롬프트 vs 나쁜 프롬프트 비교
6. 키워드 3: 문서화 (Documentation) - "AI가 바꾼 문서화의 경제학"
7. 3가지 키워드 요약

**각 키워드 대학생 맞춤 예시**:
- 선택: "전공 선택, 진로 결정에서 AI의 역할은 정보 제공, 선택은 나"
- 계획: "과제 프롬프트도 구체적으로 → 결과물 퀄리티 상승"
- 문서화: "팀프로젝트 회의록, 발표 자료를 AI가 자동 생성"

---

### Part 4: 일과 삶의 전환 - 실전 실습 (35분)

**목표**: 대학생활에 바로 적용할 수 있는 실전 경험

**슬라이드 구성** (7장):
1. Part 4 타이틀
2. 실습 4-1: 학업 도우미 소개
3. 실습 4-1: 과제 리서치 & 정리 (15분)
4. 실습 4-2: 나만의 도구 만들기 소개
5. 실습 4-2: 바이브 코딩 (20분)
6. 실전 사례: AI Agent 프로젝트 7가지
7. 대학생이 AI Agent를 쓰면 달라지는 것들

**실습 4-1: 학업 도우미 (15분)**
- 과제 주제에 대한 리서치 정리
- "이 주제에 대한 리포트 개요를 작성하고, 참고할 키워드와 구조를 정리해줘"
- 결과물: research_outline.md

**실습 4-2: 바이브 코딩 - 나만의 도구 (20분)**
- 3개 템플릿 중 선택:
  1. 시간표 관리 페이지 (가장 단순)
  2. 할일 목록 (To-Do) 앱 (중간)
  3. 팀프로젝트 투표 페이지 (조금 복잡)
- "개발자가 아니어도 만들 수 있다!" 체감

---

### Part 5: 마무리 & 다음 단계 (15분)

**목표**: 현실적 기대치 설정 + 지속 학습 동기 부여

**슬라이드 구성** (4장):
1. Part 5 타이틀
2. Agent의 한계와 가능성 (할 수 있는 것 vs 못하는 것)
3. AI 시대 대학생의 경쟁력 (3가지 조언)
4. 심화 학습 로드맵 + 감사 슬라이드

**3가지 조언**:
1. "도구를 두려워하지 마세요" - AI Agent는 협업 파트너
2. "선택하는 능력을 키우세요" - AI가 만들어도 판단은 사람의 몫
3. "기록하는 습관을 들이세요" - AI가 문서화 비용을 0에 가깝게 만들었다
