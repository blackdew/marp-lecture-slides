# Just Plan It!! - AI Agent로 시작하는 일과 삶의 전환

**강의 제목**: Just Plan It!! - AI Agent로 시작하는 일과 삶의 전환
**대상**: 동명대학교 대학생, 대학원생, 교직원
**시간**: 2시간 (120분)
**형식**: 이론 30% + 시연/실습 50% + Q&A 20%

---

## 강의 핵심 목표

**한 줄 요약**: AI Agent를 직접 체험하고, 각자의 역할(학생/대학원생/교직원)에 맞게 AI와 협업하는 관점을 장착한다

**3가지 핵심 역량**:

1. **AI Agent 체험**: 챗봇을 넘어선 AI Agent의 실체를 직접 경험
2. **AI 시대 관점 전환**: 선택, 계획, 문서화 - 3가지 키워드로 새로운 시각 장착
3. **일과 삶의 적용**: 학업/연구/업무에 AI Agent 활용법 체득

---

## 대상 분석

### 페르소나 1: 학부생 "김지훈" (컴퓨터공학과 3학년)
- AI 경험: 초급~중급 (ChatGPT 일상적 사용)
- 기대사항: 과제/프로젝트 효율화, 졸업 전 실무 경험
- 관심사: 학업 효율화 80%, 취업 역량 60%

### 페르소나 2: 대학원생 "박서연" (산업공학 석사 2학기)
- AI 경험: 중급 (ChatGPT/Claude 유료 구독)
- 기대사항: 논문 탐색/정리 자동화, 연구 워크플로우 효율화
- 관심사: 연구 워크플로우 90%, 업무 자동화 20%

### 페르소나 3: 교직원 "이민수" (행정팀 주임, 7년차)
- AI 경험: 초급 (ChatGPT 기본 질의응답)
- 기대사항: 반복 행정 업무 감소, 회의록/문서 자동화
- 관심사: 업무 자동화 100%

### 전체 수강생 특성
- AI 경험 수준 분포: 초급 40% / 중급 45% / 고급 15%
- 핵심 니즈: 역할별 실용적 활용법 (학업/연구/업무)
- 난이도 배분: 기본 50% / 중급 40% / 고급 10% (선택)

## Part별 상세 구성

### Part 1: 오프닝 - AI Agent, 왜 지금인가? (15분)

**목표**: 흥미 유발 + AI Agent 개념 이해 (3개 역할 모두 공감할 수 있는 출발점)

**슬라이드 구성** (4장):
1. 타이틀 슬라이드 (강의 제목, QR코드)
2. 오늘의 여정 (전체 타임라인)
3. 챗봇 vs Agent 비교 (표 + 비유)
4. Agent 구조: LLM + Memory + Planning + Tools

**핵심 메시지**: "Agent = 도구를 쓸 줄 아는 AI 동료"

**오프닝 훅 (범용화)**:
- "지금 이 순간, 여러분의 업무/학업에서 가장 반복적인 작업은 무엇인가요?"
- "만약 AI가 그 작업을 대신 해준다면?"
- 예시:
  - 학생: "매주 리포트 작성, 자료 검색"
  - 대학원생: "논문 탐색, 문헌 정리"
  - 교직원: "회의록 작성, 데이터 정리"

---

### Part 2: 첫 체험 - Agent와 함께하기 (20분)

**목표**: 성공 경험을 통한 "와, 이게 되네!" 체감

**슬라이드 구성** (5장):
1. CLI 도구 비교 (Claude Code / Gemini / Codex)
2. 설치 및 첫 실행
3. 실습 2-1: 자기소개 페이지 만들기 (HTML)
4. 실습 2-2: Plan 모드 체험
5. 중간 회고: "어디에 써먹을 수 있을까?"

**실습 2-1: 자기소개 페이지 만들기 (범용적)**
- 프롬프트: "나를 소개하는 HTML 페이지를 만들어줘"
- 포함 내용: 이름, 소속/전공, 역할(학생/대학원생/교직원), 관심사
- 브라우저에서 열어보기

**실습 2-2: Plan 모드 체험**
- Shift+Tab 두 번 → Plan 모드 진입
- "자기소개 페이지에 다크모드 토글 추가해줘"
- 계획 확인 → 승인 → 실행

**핵심 포인트**:
- 이 실습은 이미 범용적 (역할 무관)
- 코딩 경험 불필요 (접근성 최대화)

---

### Part 3: AI 시대 3가지 키워드 (25분)

**목표**: 체험을 개념으로 정리 + 관점 전환

**슬라이드 구성** (7장):
1. Part 3 타이틀
2. 키워드 1: 선택 (Choice) - "AI가 해도 결정은 당신의 몫"
3. LLM 작동 원리 (확률 기반 예측)
4. 키워드 2: 계획 (Planning) - "좋은 프롬프트 = 좋은 계획"
5. 좋은 프롬프트 vs 나쁜 프롬프트 비교
6. 키워드 3: 문서화 (Documentation) - "AI가 바꾼 문서화의 경제학"
7. 3가지 키워드 요약

**각 키워드 3-track 맞춤 예시**:

**키워드 1: 선택 (Choice)**
- **학부생**: "전공 선택, 진로 결정에서 AI는 정보 제공, 선택은 나"
- **대학원생**: "논문 주제, 연구 방향성에서 AI는 옵션 제시, 판단은 연구자"
- **교직원**: "업무 처리 방식, 문서 최종 승인에서 AI는 초안, 결정은 담당자"

**키워드 2: 계획 (Planning)**
- **학부생**: "과제 프롬프트도 구체적으로 → 결과물 퀄리티 상승"
- **대학원생**: "논문 검색 쿼리를 명확히 → 관련도 높은 결과 확보"
- **교직원**: "문서 작성 요구사항을 상세히 → 수정 횟수 감소"

**키워드 3: 문서화 (Documentation)**
- **학부생**: "팀프로젝트 회의록, 발표 자료를 AI가 자동 생성"
- **대학원생**: "실험 노트, 연구 일지를 AI가 구조화"
- **교직원**: "회의록, 보고서 초안을 AI가 즉시 작성"

---

### Part 4: 일과 삶의 전환 - 실전 실습 (35분)

**목표**: 역할별로 바로 적용할 수 있는 실전 경험

**슬라이드 구성** (8장):
1. Part 4 타이틀
2. 실습 4-1 소개: 역할별 시나리오
3. 실습 4-1: 학생 트랙 (과제 리서치 & 정리) - 10분
4. 실습 4-1: 대학원생 트랙 (논문 탐색 & 요약) - 10분
5. 실습 4-1: 교직원 트랙 (회의록 자동 작성) - 10분
6. 실습 4-2 소개: 나만의 도구 만들기 (공통)
7. 실습 4-2: 바이브 코딩 - 15분
8. 실전 사례: AI Agent 프로젝트 7가지

**실습 4-1: 역할별 시나리오 분기 (각 10분, 수강생은 자신의 역할 선택)**

**학생 트랙: 학업 도우미**
- 프롬프트: "다음 과제 주제에 대한 리포트 개요를 작성하고, 참고할 키워드와 구조를 정리해줘"
- 결과물: `research_outline.md`
- 추가 실습: "참고문헌 5개를 검색하고 요약해줘"

**대학원생 트랙: 논문 탐색 & 정리**
- 프롬프트: "내 연구 주제는 [주제명]이야. 관련 논문 5편을 검색하고 각각 한 줄로 요약해줘"
- 도구 소개: SciSpace, Connected Papers, Consensus
- 결과물: `paper_summary.md`

**교직원 트랙: 회의록 자동 작성**
- 프롬프트: "다음 회의 내용을 회의록으로 정리해줘 (참석자, 안건, 결정사항, 할 일)"
- 도구 소개: 클로바노트 (월 600분 무료)
- 결과물: `meeting_minutes.md`

**실습 4-2: 바이브 코딩 - 나만의 도구 (공통, 15분)**
- 3개 템플릿 중 선택:
  1. **시간표 관리 페이지** (학생/교직원 모두 가능)
  2. **할일 목록 (To-Do) 앱** (범용)
  3. **팀프로젝트 투표 페이지** (학생/교직원)
- "개발자가 아니어도 만들 수 있다!" 체감

**실전 사례 (역할별 1-2개씩)**:
- 학생: 발표자료 자동 생성 (AiPPT, 미리캔버스)
- 대학원생: Deep Research로 리서치 시간 80% 단축
- 교직원: MCP 기반 복합 업무 자동화 (이메일+캘린더+문서)

### Part 5: 마무리 & 다음 단계 (15분)

**목표**: 현실적 기대치 설정 + 지속 학습 동기 부여

**슬라이드 구성** (4장):
1. Part 5 타이틀
2. Agent의 한계와 가능성 (할 수 있는 것 vs 못하는 것)
3. AI 시대 경쟁력을 위한 3가지 조언 (범용화)
4. 심화 학습 로드맵 + 감사 슬라이드

**AI 시대 경쟁력을 위한 3가지 조언 (범용화)**:

1. **"도구를 두려워하지 마세요"**
   - 학생: AI Agent는 학업의 협업 파트너
   - 대학원생: AI Agent는 연구의 가속기
   - 교직원: AI Agent는 업무의 효율화 도구

2. **"선택하는 능력을 키우세요"**
   - 학생: AI가 만든 리포트도 스스로 검증하고 선택
   - 대학원생: AI가 제시한 논문도 직접 읽고 판단
   - 교직원: AI가 작성한 문서도 최종 검토와 승인은 본인

3. **"기록하는 습관을 들이세요"**
   - 공통: AI가 문서화 비용을 0에 가깝게 만들었다
   - 학생: 학습 일지, 프로젝트 회고
   - 대학원생: 실험 노트, 연구 일지
   - 교직원: 회의록, 업무 로그

**심화 학습 로드맵**:
- 무료 도구 탐색: Claude, Gemini, OpenClaw
- 대학원생용 도구: SciSpace, Connected Papers, Consensus
- 교직원용 도구: 클로바노트, 마음회의록
- MCP 표준 학습 (2026년 산업 표준)
