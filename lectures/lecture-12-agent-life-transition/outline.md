# AI Agent 활용 생활/업무 전환

**강의 제목**: AI Agent 활용 생활/업무 전환
**대상**: 동명대학교 대학생, 대학원생, 교직원
**시간**: 2시간 (120분)
**형식**: Opening Hook + 4-Act 구조 (시연 + 직접 실습 중심)

---

## 강의 핵심 목표

**한 줄 요약**: Claude Desktop GUI를 활용한 실습 중심 강의로, 학생/대학원생/교직원 각자의 삶과 업무에 AI Agent를 적용하는 관점 전환을 경험한다

**3가지 핵심 역량**:

1. **Agent 믿음**: Opening Hook에서 회의록 자동 정리를 목격하며 Agent에 대한 믿음을 형성
2. **일 맡기기**: 시킬지 고민하고, 좋은 프롬프트를 작성하며, Agent와 협업하는 방법 습득
3. **실천 시작**: 오늘부터 시작할 수 있는 구체적 실천 방법과 동기 획득

---

## 대상 분석

### 페르소나 1: 김민지 (대학생, 경영학과 3학년)

**배경**: 비전공자, 코딩 경험 전무. 팀 프로젝트, 보고서 작성, 발표 자료 제작이 주요 업무.
**AI 경험 수준**: 초급 (ChatGPT 기본 대화형 사용만 가능)
**기대사항**: 과제 시간 절약, 팀플 자료 정리/발표 준비 효율화, 바로 따라할 수 있는 방법
**이탈 위험**: 이론 10분+ 시 집중력 저하, CLI/코드 등장 시 "나랑 상관없는 내용", 추상적 사례 외면
**고충 → AI 해결**: 팀플 자료 정리/발표 준비/과제 초안 → 문서 요약, 아웃라인 제안, 이미지 검색 자동화

### 페르소나 2: 이준호 (대학원생, 컴퓨터공학 석사)

**배경**: 코딩 가능 (Python, Git). 논문 탐색, 실험 데이터 분석이 주요 업무. ChatGPT Plus + Copilot 사용.
**AI 경험 수준**: 중급 (코딩 보조 도구 활용, 프롬프트 엔지니어링 기초 이해)
**기대사항**: 논문 100편 빠르게 스캔, 실험 데이터 분석 자동화, 실제로 돌아가는 것
**이탈 위험**: 기초 ChatGPT 설명 길면 지루, "이미 아는 내용" 느낌, 시연만 보고 끝나면 허탈
**고충 → AI 해결**: 논문 PDF 다량 처리/실험 결과 요약 → 파일 일괄 처리 Agent, 데이터 분석 자동화

### 페르소나 3: 박영희 (교직원, 학과 행정팀)

**배경**: 비개발자, Excel/한글/PPT 주요 도구. AI 도구 사용 경험 거의 없음.
**AI 경험 수준**: 입문 (AI 챗봇 사용 경험 전무)
**기대사항**: 반복 문서 작성 자동화, 학생 데이터 정리 시간 절약, 당장 내일부터 쓸 수 있는 것
**이탈 위험**: 설치/설정 복잡하면 즉시 이탈, 전문 용어 혼란, "개발자 이야기" 느낌 시 관심 상실
**고충 → AI 해결**: 공문서 작성/Excel 정리/회의록 작성 → 템플릿 문서 생성, 표 자동 분류, 음성→텍스트

### 전체 수강생 특성

- **AI 경험 수준 분포**: 입문 30% (박영희) / 초급 50% (김민지) / 중급 20% (이준호)
- **핵심 니즈**: 역할별 실용적 활용법 (학업/연구/업무)
- **난이도 배분**: 기본 60% / 중급 30% / 고급 10%
