# AI 에이전트로 자동화 시작하기: CLI 실습으로 배우는 기초 활용법

## 강의 개요

**대상:** 완전 초보자 (Agent 처음 접함)
**시간:** 120분
**키워드:** 선택, 계획, 문서화

### 핵심 메시지

이 강의는 단순한 CLI 도구 사용법을 넘어, AI 시대를 살아가는 철학적 관점을 제시합니다.

- **선택**: AI가 모든 것을 할 수 있어도, 최종 선택은 인간의 몫입니다
- **계획**: 컨텍스트를 채우는 것이 실행보다 중요합니다
- **문서화**: 문서는 프로젝트의 진짜 결과물입니다
