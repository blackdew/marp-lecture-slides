# Part 2: 첫 업무 시켜보기 (JavaScript 기초 with Claude)

## Part 2: Do → See → Ask → Learn

### 이번 시간의 흐름

| 단계 | 활동 | 시간 |
|------|------|------|
| **Do** | 자기소개 페이지, 웹 계산기 만들기 | 25분 |
| **See** | 실행 결과 확인 | 5분 |
| **Ask** | 모르는 코드 질문하기 | 10분 |
| **Learn** | 핵심만 정리 | 5분 |

**핵심 메시지**: "신입 인턴(AI)에게 작은 업무부터 맡겨보며 호흡을 맞춰봅시다."

---

## [Do] 실습 2-1: 자기소개 웹페이지

### Claude에게 요청

```
나의 자기소개 웹페이지를 만들어줘.

이름: [본인 이름]
직업/소속: [본인 직업]
관심사: [2-3가지]
한 줄 소개: [자유롭게]

예쁜 디자인으로 만들어줘. 파일명은 about-me.html로 해줘.
```

### 브라우저에서 열기

파일 탐색기에서 `about-me.html`을 더블클릭하거나:

```bash
open about-me.html        # Mac
start about-me.html       # Windows
```

> **와!** 몇 초 만에 웹페이지가 완성되었습니다!

---

## [See] 실습 2-1: 결과 확인

### 확인할 것

| 항목 | 확인 |
|------|------|
| 파일 생성됨? | `about-me.html` 파일 존재 |
| 브라우저에서 열림? | 예쁜 페이지가 보임 |
| 내 정보가 있음? | 이름, 직업, 관심사 표시 |

### 이게 Agent의 힘입니다

- HTML 코드 작성? → Agent가 함
- CSS 스타일링? → Agent가 함
- **당신이 한 것**: 원하는 걸 말했을 뿐

> **"상사(나)는 기획하고, 인턴(AI)은 구현합니다."**

## [Do] 실습 2-2: 웹 계산기

### Claude에게 요청

```
간단한 계산기 웹페이지를 만들어줘.

- 숫자 두 개를 입력하는 칸
- 사칙연산(+, -, ×, ÷) 버튼
- 결과 표시 영역
- 예쁜 디자인

파일명은 calculator.html로 해줘.
```

### 브라우저에서 열기

```bash
open calculator.html        # Mac
start calculator.html       # Windows
```

---

## [See] 실습 2-2: 결과 확인

### 확인할 것

| 항목 | 확인 |
|------|------|
| 브라우저에서 열림? | 계산기 화면이 보임 |
| 숫자 입력됨? | 두 칸에 숫자 입력 가능 |
| 버튼 동작? | 연산 버튼 클릭 시 결과 표시 |

### 궁금한 점 메모

- `<button>`이 뭐지?
- `onclick`이 뭐지?
- JavaScript 함수는 어떻게 동작해?

> **이제 AI에게 질문할 시간입니다!**

---

## [Ask] AI에게 질문하기

### 이렇게 물어보세요

```
난 프로그램을 전혀 몰라. 나에게 작성된 코드에 대해 설명해줘
```

```
about-me.html 코드에서 <style> 태그가 뭐야?
```

```
calculator.html에서 onclick은 어떻게 작동해?
```

### 질문 팁

- 코드를 복사해서 붙여넣고 질문하면 더 정확한 답변!
- "왜"보다 "어떻게"로 시작하면 더 실용적인 답변!

---

## [Ask] 자주 나오는 질문들

### HTML/CSS 관련 (실습 2-1)

| 질문 | 핵심 답변 |
|------|-----------|
| "HTML이 뭐야?" | 웹페이지의 구조/내용 |
| "CSS가 뭐야?" | 웹페이지의 스타일/디자인 |
| "\<div\>가 뭐야?" | 내용을 묶는 상자 |

### JavaScript 관련 (실습 2-2)

| 질문 | 핵심 답변 |
|------|-----------|
| "onclick이 뭐야?" | 클릭 시 실행되는 이벤트 |
| "function이 뭐야?" | 재사용 가능한 코드 묶음 |
| "document.getElementById?" | HTML 요소 찾는 함수 |

## [Learn] 이번 시간에 배운 핵심

### 실습하면서 배운 것

| 실습 | 배운 것 | 핵심 |
|------|---------|------|
| **자기소개** | HTML/CSS | 구조 + 디자인 |
| **웹 계산기** | HTML/CSS/JS | 구조 + 동작 |

### Agent와 함께하는 학습법

1. **만들어 달라고 요청** → Agent가 코드 생성
2. **실행해서 결과 확인** → 동작하는지 체크
3. **모르는 건 질문** → Agent가 설명

> **"이 코드가 왜 이런지 이해해야, 잘못되거나 다르게 만들어진 부분을 확인할 수 있습니다."**
