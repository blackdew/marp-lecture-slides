### Part 1: AI Agent 소개 + 환경 설정 (50분)

**목표**: AI Agent 개념 이해 및 Gemini CLI 환경 구축

**내용**:

- **AI Agent 기초** (15분):
  - 챗봇 vs 에이전트: 말만 하는 AI vs 행동하는 AI
  - Agent 구조: LLM + Memory + Planning + Tools
  - "생각만 하던 AI에게, 실행하는 손발이 생겼습니다"

- **환경 설정** (25분):
  - **Node.js 확인**: `node --version` (Gemini CLI 설치 시 이미 설치됨)
  - **Gemini CLI 설치**: `npm install -g @google/gemini-cli`
  - Google 계정 인증 및 첫 실행
  - 터미널 기본 사용법 (cd, dir/ls 등)

- **첫 대화** (10분):
  - "안녕, 너는 누구니?" → Gemini 응답 확인
  - 간단한 파일 생성 요청 체험

**대안 안내**: AntiGravity (GUI 기반) - CLI가 어려운 참가자용

### Part 3: 파일 자동화 실습 + AI 시대 우리의 역할 (90분)

**목표**: CSV 처리 및 파일 정리 자동화, AI 시대 우리 역할 이해

**내용**:

- **Node.js 파일 처리 개념** (10분):
  - fs 모듈: 파일 읽기/쓰기
  - csv-parser: CSV 파일 처리
  - 패키지 설치: `npm install csv-parser csv-writer`

- **실습 3-1: CSV 데이터 통합** (25분) ⭐⭐
  - 샘플: `sales_jan.csv`, `sales_feb.csv`, `sales_mar.csv`
  - 프롬프트: "세 개의 CSV 파일을 하나로 합치고, 날짜 포맷을 통일해줘"
  - 결과: `sales_merged.csv`
  - **도전**: 담당자별 매출 합계 추가

- **실습 3-2: 폴더 자동 정리 스크립트** (25분) ⭐
  - 시나리오: Downloads 폴더의 파일들을 확장자별로 분류
  - 프롬프트: "폴더 내 파일들을 확장자별로 분류하는 스크립트 만들어줘"
  - fs, path 모듈 이해
  - **도전**: 날짜별 폴더 생성

- **AI 시대 우리의 역할** (20분):
  - 실습 경험을 바탕으로 이론 정리
  - **선택**: Agent가 비교 제공 → 나의 가치로 선택
  - **계획**: 비어있는 컨텍스트 → 계획에 70% 시간 투자
  - **문서화**: Agent로 거의 무료 → 문서 = 프로젝트
  - "AI가 만들어줘도, 선택·계획·문서화는 우리의 몫"

**산출물**:
- ✅ 프로토타입 1: 폴더 정리 스크립트
- ✅ 프로토타입 2: CSV 통합 분석기

### Part 5: 나만의 프로토타입 만들기 (70분)

**목표**: Express로 실용적인 웹앱 프로토타입 제작

**내용**:

- **Express 소개** (10분):
  - "Node.js로 웹서버를 쉽게 만드는 프레임워크"
  - 설치: `npm install express`
  - 실행: `node app.js` → `http://localhost:3000`

- **실습 5-1: 기본 웹앱 만들기** (15분):
  - 프롬프트: "제목과 입력창이 있는 간단한 웹페이지를 Express로 만들어줘"
  - HTML, 폼, 라우팅 이해

- **해커톤: 프로토타입 제작** (35분):
  - **3개 템플릿 중 선택**:
    1. **보고서 자동 생성기**: 주제 입력 → 개요/본문 생성
    2. **수업 노트 요약기**: 텍스트 입력 → 요약 + 퀴즈 생성
    3. **취업 공고 비교 도구**: 정보 입력 → 비교표 생성

  - 기본 기능 구현 (25분) + 개선 (10분)

- **발표 및 공유** (10분):
