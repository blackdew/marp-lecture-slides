## 🧠 AI 시대 학습 철학

### 기존 방식 vs Agent 시대 방식

| 구분 | 기존 방식 | Agent 시대 방식 |
|------|----------|----------------|
| 순서 | 이론 → 실습 | **실습 → 결과 확인 → 질문** |
| 학습 | 개념 이해 후 적용 | **먼저 해보고, 모르는 건 물어보기** |
| 교사 | 강사가 설명 | **AI가 설명 (개인 맞춤)** |
| 속도 | 전체가 같은 속도 | **각자 다른 속도** |

### 실습 사이클: Do → See → Ask

```
1. Do (실습)     : 프롬프트로 요청, 결과물 생성
2. See (확인)    : 결과물 실행, 동작 확인
3. Ask (질문)    : 모르는 부분을 AI에게 질문
4. Learn (정리)  : 핵심 개념만 간단히 정리
```

> **"먼저 해보고, 결과를 보고, 모르면 물어보세요. AI가 설명해줍니다."**

### Part 1: AI Agent 소개 + 환경 설정 (50분)

**목표**: AI Agent 개념 이해 및 Gemini CLI 환경 구축

**내용**:

- **AI Agent 기초** (15분):
  - 챗봇 vs 에이전트: 말만 하는 AI vs 행동하는 AI
  - Agent 구조: LLM + Memory + Planning + Tools
  - "생각만 하던 AI에게, 실행하는 손발이 생겼습니다"

- **환경 설정** (25분):

  - Google 계정 인증 및 첫 실행
  - 터미널 기본 사용법 (cd, dir/ls 등)

- **첫 대화** (10분):
  - "안녕, 너는 누구니?" → Gemini 응답 확인
  - 간단한 파일 생성 요청 체험

**대안 안내**: AntiGravity (GUI 기반) - CLI가 어려운 참가자용

### Part 2: JavaScript 기초 with Gemini (45분)

**목표**: Gemini가 생성한 JavaScript 코드를 실행하고, 질문으로 이해하기

**학습 방식**: Do → See → Ask → Learn

- **Do: 실습 2-1 Hello World** (10분):
  - 프롬프트: "Hello World 출력하는 JavaScript 코드 만들어서 hello.js로 저장해줘"
  - 실행: `node hello.js`

- **See: 결과 확인** (5분):
  - 파일이 생성되었는가?
  - 실행 결과가 나오는가?

- **Ask: 질문으로 학습** (10분):
  - "이 코드에서 console.log가 뭐야?"
  - "const랑 let은 뭐가 달라?"
  - "세미콜론은 왜 붙여?"

- **Do: 실습 2-2 계산기** (15분):
  - 프롬프트: "두 숫자를 입력받아 더하는 프로그램 만들어줘"
  - 실행 후 결과 확인
  - 궁금한 부분 질문: "readline이 뭐야?", "parseInt는 왜 써?"

- **Learn: 핵심 정리** (5분):
  - 오늘 질문한 것 중 핵심만 정리
  - **도전**: 사칙연산 계산기로 확장

**핵심 메시지**: "코드를 외우지 마세요. 모르면 AI에게 물어보세요."

### Part 3: 파일 자동화 실습 + AI 시대 우리의 역할 (90분)

**목표**: CSV 처리 및 파일 정리 자동화, AI 시대 우리 역할 이해

**학습 방식**: Do → See → Ask → Learn

- **준비** (5분):
  - 패키지 설치: `npm install csv-parser csv-writer`
  - 샘플 파일 확인: `samples/part-03-csv/`

- **Do: 실습 3-1 CSV 데이터 통합** (15분) ⭐⭐
  - 프롬프트: "sales_jan.csv, sales_feb.csv, sales_mar.csv를 합치고 날짜를 YYYY-MM-DD로 통일해줘"
  - 실행: `node merge_csv.js`

- **See: 결과 확인** (5분):
  - `sales_merged.csv` 생성 확인
  - 날짜 형식이 통일되었는가?

- **Ask: 질문으로 학습** (10분):
  - "fs.readFileSync가 뭐야?"
  - "csv-parser는 어떻게 작동해?"
  - "async/await는 왜 쓰는 거야?"

- **Do: 실습 3-2 폴더 자동 정리** (15분) ⭐
  - 프롬프트: "part-03-organize 폴더의 파일들을 확장자별로 분류해줘"
  - 실행 후 결과 확인

- **Ask: 질문으로 학습** (10분):
  - "path.extname이 뭐야?"
  - "fs.mkdirSync의 recursive 옵션은?"
  - 궁금한 거 자유롭게 질문

- **Learn: AI 시대 우리의 역할** (20분):
  - 실습 경험을 바탕으로 성찰
  - **선택**: Agent가 비교 제공 → 나의 가치로 선택
  - **계획**: 컨텍스트 채우기 → 계획에 70% 시간 투자
  - **문서화**: Agent로 거의 무료 → 문서 = 프로젝트

- **도전 과제** (10분):
  - CSV: 담당자별 매출 합계 추가
  - 폴더: 날짜별 하위 폴더 생성

**산출물**:
- ✅ 프로토타입 1: 폴더 정리 스크립트
- ✅ 프로토타입 2: CSV 통합 분석기

**목표**: 웹 크롤링 기초 및 AI 콘텐츠 생성 체험

  - 윤리적 고려사항 (저작권, 개인정보)

- **도전 과제** (5분):
  - 여러 페이지 크롤링
  - 1초 간격으로 요청하기

### Part 5: 나만의 프로토타입 만들기 (70분)

**목표**: Express로 실용적인 웹앱 프로토타입 제작

**학습 방식**: Do → See → Ask → Learn

- **Do: 실습 5-1 기본 웹앱** (15분):
  - 패키지 설치: `npm install express`
  - 프롬프트: "제목과 입력창이 있는 간단한 웹페이지를 Express로 만들어줘"
  - 실행: `node app.js` → `http://localhost:3000`

- **See: 결과 확인** (5분):
  - 브라우저에서 페이지가 보이는가?
  - 입력창이 동작하는가?

- **Ask: 질문으로 학습** (5분):
  - "Express가 뭐야?"
  - "app.get이랑 app.post 차이가 뭐야?"
  - "localhost:3000은 뭐야?"

- **Do: 해커톤 - 프로토타입 제작** (35분):
  - **3개 템플릿 중 선택**:
    1. **보고서 자동 생성기**: 주제 입력 → 개요/본문 생성
    2. **수업 노트 요약기**: 텍스트 입력 → 요약 + 퀴즈 생성
    3. **취업 공고 비교 도구**: 정보 입력 → 비교표 생성
  - 막히면 AI에게 질문하면서 진행
  - **팁**: "이거 왜 안 돼?" → 에러 메시지 붙여넣기

- **발표 및 공유** (10분):
  - 각자 만든 앱 시연 (2분)
  - "만들면서 뭘 물어봤어요?" 공유
  - 베스트 프로토타입 선정

**산출물**:
- ✅ 프로토타입 4: 보고서 자동 생성기 (또는 수업 노트 요약기)
