# 코딩 없는 데이터 분석

- **No-Code Data Analysis**: 파이썬/R 코드를 몰라도 자연어로 데이터 분석 가능
- **Gemini의 역할**:
    - 데이터 이해 및 구조 파악
    - 분석 코드(Python) 자동 생성 및 실행 (AntiGravity 내장 기능 활용)
    - 인사이트 도출

---

# 실습 1: 데이터 로드 및 파악

- **데이터**: `sales_data.csv` (월별 매출, 품목, 지역 등 포함)
- **지시**:
    > "이 데이터를 분석해줘. 어떤 컬럼들이 있고, 데이터의 전반적인 특징은 뭐야?"

- AntiGravity가 데이터를 읽고 요약 정보를 제공합니다.

---

# 실습 2: 인사이트 도출 및 시각화

- **지시**:
    > "가장 매출이 높은 지역은 어디야? 그리고 월별 매출 추이를 꺾은선 그래프로 그려줘."

- **결과**:
    - 텍스트로 된 분석 결과
    - 생성된 그래프 이미지 또는 차트

# 실습 3: Google Sheets 연동

- **확장**: 분석된 데이터를 Google Sheets로 정리
- **지시**:
