# 실습 2: PDF/문서 요약 및 통합

## 목표

여러 보고서 문서를 읽고 자동으로 요약하거나 통합하기

## 샘플 파일 위치

```
samples/practice-02-pdf/
├── report1.txt           # 영업 실적 보고서
├── report2.txt           # 개발팀 분기 보고서
└── invoice_table.txt     # 세금계산서
```

> 실제로는 PDF 파일이지만, 실습 편의를 위해 .txt로 제공

## 실습 과제

### 1단계: 단일 보고서 요약

**프롬프트 예시:**
```
samples/practice-02-pdf/report1.txt 파일을 읽고 다음 형식으로 요약해줘:

## 보고서 요약

**작성자:**
**보고 기간:**

**핵심 내용 (3-5줄):**

**주요 수치:**
-
-

**액션 아이템:**
-

결과를 report1_summary.md로 저장해줘.
```

**기대 결과:**
- 구조화된 마크다운 요약본
- 핵심 수치 추출
- 액션 아이템 정리

### 2단계: 여러 보고서 통합 요약

**프롬프트 예시:**
```
samples/practice-02-pdf/ 폴더의 report1.txt와 report2.txt를 읽고:

1. 각 보고서의 핵심 내용 요약
2. 공통 주제 찾기
3. 전체 통합 요약본 작성
4. 팀별 주요 성과 비교표 작성

결과를 quarterly_summary.md로 저장해줘.
```

**기대 결과:**
```markdown
# 2024년 1분기 통합 보고서

## 팀별 요약

### 영업팀
- 총 매출: 15억원
- 신규 고객: 45개사

### 개발팀
- 완료 프로젝트: 2건
- 진행 프로젝트: 2건

## 공통 이슈
...
```

### 3단계: 표 데이터 추출

**프롬프트 예시:**
```
samples/practice-02-pdf/invoice_table.txt 파일에서:

1. 품목 내역 표를 CSV로 변환
2. 합계 금액 검증
3. 결과를 invoice_items.csv로 저장

CSV는 다음 컬럼으로: 품목명,수량,단가,공급가액
```

**기대 결과:**
- 정확한 표 데이터 추출
- CSV 형식 변환
- 금액 계산 검증
