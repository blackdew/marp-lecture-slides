# 해커톤 실습 과제 가이드

이 폴더에는 AI 에이전트 해커톤을 위한 4가지 실습 과제가 준비되어 있습니다.

## 실습 과제 목록

### 1. CSV/엑셀 정리 자동화
**난이도:** ⭐⭐ (초급~중급)
**소요시간:** 20-30분
**핵심 스킬:** 데이터 정제, 포맷 통일, 파일 통합

→ [상세 가이드](./practice-01-csv-cleanup.md)

### 2. PDF/문서 요약 및 통합
**난이도:** ⭐⭐ (초급~중급)
**소요시간:** 20-30분
**핵심 스킬:** 텍스트 분석, 요약, 구조화

→ [상세 가이드](./practice-02-pdf-summary.md)

### 3. 메일/공지 자동 생성
**난이도:** ⭐⭐⭐ (중급)
**소요시간:** 25-35분
**핵심 스킬:** 콘텐츠 생성, 톤 조절, 구조화

→ [상세 가이드](./practice-03-email-generation.md)

### 4. 파일 정리 자동화
**난이도:** ⭐⭐⭐ (중급)
**소요시간:** 30-40분
**핵심 스킬:** 파일 시스템 조작, 패턴 인식, 스크립트 생성

→ [상세 가이드](./practice-04-file-organizer.md)

## 실습 진행 방법

### 준비 단계

1. 샘플 파일 확인
   ```bash
   cd lectures/lecture-08-hackathon-practice/samples
   ls -R
   ```

2. Claude Code 실행
   ```bash
   code
   ```

3. 작업 폴더로 이동
   ```bash
   cd lectures/lecture-08-hackathon-practice/samples
   ```

### 실습 단계

1. 각 실습 가이드의 프롬프트 예시 참고
2. 본인의 언어로 자연스럽게 재작성
3. Claude에게 요청
4. 결과 확인 및 개선
5. 다음 단계로 진행

### 성공 기준

각 실습의 체크리스트를 모두 통과하면 성공입니다.

## 팀 해커톤 진행 시

### 역할 분담 예시

- **프롬프트 엔지니어:** 요구사항 정리 및 프롬프트 작성
- **검증자:** 결과물 검증 및 개선사항 도출
- **문서화 담당:** 과정 및 결과 정리

### 평가 기준

1. **완성도** (40점)
   - 요구사항 충족 여부
   - 결과의 정확성

2. **효율성** (30점)
   - 프롬프트의 명확성
   - 작업 속도

3. **창의성** (30점)
   - 독창적인 접근 방법
   - 추가 기능 구현

## 주의사항

- 실제 파일 시스템 조작이 필요한 4번 실습은 테스트 폴더에서 진행하세요
- 스크립트 실행 전 항상 백업을 만드세요
- 프롬프트는 예시일 뿐, 자유롭게 수정하세요

## 도움말

2. 단계를 더 작게 나누어보세요
3. 예시 출력 형식을 명확히 제시하세요
