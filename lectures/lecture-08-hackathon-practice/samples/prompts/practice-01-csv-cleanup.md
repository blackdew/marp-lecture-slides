# 실습 1: 엑셀/CSV 정리 자동화

## 목표

엉망진창인 CSV 파일들을 자동으로 정리하고 통합하는 작업

## 샘플 파일 위치

```
samples/practice-01-csv/
├── sales_jan.csv          # 1월 매출 데이터 (날짜/금액 포맷 불일치)
├── sales_feb.csv          # 2월 매출 데이터 (컬럼명 다름)
├── sales_mar.csv          # 3월 매출 데이터 (한글 컬럼명)
└── customer_data_messy.csv # 고객 데이터 (중복 있음)
```

## 실습 과제

### 1단계: 월별 매출 파일 통합

**프롬프트 예시:**
```
samples/practice-01-csv/ 폴더에 있는 sales_jan.csv, sales_feb.csv, sales_mar.csv 파일을 읽어서 다음 작업을 수행해줘:

1. 컬럼명을 통일해서 (날짜, 제품명, 수량, 금액, 담당자)
2. 날짜 포맷을 모두 YYYY-MM-DD 형식으로 통일
3. 금액에서 "원", "₩", 쉼표 같은 문자 제거하고 숫자만 남기기
4. 세 파일을 하나로 합치기
5. 결과를 sales_combined.csv로 저장

완료되면 어떤 작업을 했는지 요약해줘.
```

**기대 결과:**
- 통일된 포맷의 단일 CSV 파일
- 날짜: 2024-01-05 형식
- 금액: 순수 숫자 (1200000)
- 컬럼명: 한글로 통일

### 2단계: 고객 데이터 정리

**프롬프트 예시:**
```
samples/practice-01-csv/customer_data_messy.csv 파일을 정리해줘:

1. 중복 행 제거 (완전히 동일한 행만)
2. 전화번호 포맷 통일 (010-XXXX-XXXX)
3. 금액 포맷 통일 (숫자만)
4. 등급별로 정렬 (VIP > GOLD > SILVER > BRONZE)
5. 결과를 customer_data_clean.csv로 저장

정리 전후 통계를 보여줘 (총 행 수, 중복 제거 개수 등)
```

### 3단계: 피벗 테이블 생성

**프롬프트 예시:**
```
방금 만든 sales_combined.csv를 기반으로:

1. 담당자별 총 매출액 계산
2. 제품별 총 판매 수량 계산
3. 월별 매출 합계 계산
4. 결과를 sales_summary.csv로 저장

요약 통계도 함께 보여줘.
```

## 난이도 조절

**쉬운 버전:** 파일 하나만 정리하기
**중간 버전:** 두 파일 통합하기
**어려운 버전:** 모든 파일 통합 + 피벗 테이블 생성
