# 실습 3: 메일/공지 자동 생성

## 목표

회의록, 프로젝트 현황, 상품 정보 등을 기반으로 메일이나 공지문을 자동 생성

## 샘플 파일 위치

```
samples/practice-03-email/
├── meeting_notes.txt      # 주간 회의록
├── project_status.txt     # 프로젝트 현황
└── product_list.csv       # 신제품 목록
```

## 실습 과제

### 1단계: 회의록 → 액션 아이템 메일

**프롬프트 예시:**
```
samples/practice-03-email/meeting_notes.txt 파일을 읽고:

다음 형식의 팀 공지 메일을 작성해줘:

제목: [팀 공지] 주간 회의 액션 아이템 (3/15)

본문:
- 간단한 회의 요약
- 담당자별 액션 아이템 정리
- 다음 회의 일정

결과를 action_items_email.txt로 저장해줘.
```

**기대 결과:**
```
제목: [팀 공지] 주간 회의 액션 아이템 (3/15)

안녕하세요, 팀원 여러분

3월 15일 주간 회의에서 논의된 내용을 정리하여 공유드립니다.

## 주요 결정사항
- 모바일 앱 리뉴얼 프로젝트 착수
- 예산: 5,000만원 / 기간: 3개월

## 담당자별 액션 아이템

[이영희님]
- API 명세서 작성 (마감: 3/18)
- 개발 환경 셋업 가이드 작성 (마감: 3/20)

[박민수님]
...

다음 회의는 3월 22일 오전 10시입니다.

감사합니다.
```

### 2단계: 프로젝트 현황 → 경영진 보고 메일

**프롬프트 예시:**
```
samples/practice-03-email/project_status.txt 파일을 읽고:

경영진에게 보낼 간결한 프로젝트 현황 보고 메일을 작성해줘.

핵심만 강조하고, 숫자를 명확히 표시하고, 이슈는 솔루션과 함께 제시해줘.

결과를 executive_report_email.txt로 저장.
```

### 3단계: 상품 목록 → 마케팅 공지문

**프롬프트 예시:**
```
samples/practice-03-email/product_list.csv 파일을 읽고:

고객에게 보낼 신제품 출시 공지 메일을 작성해줘:

1. 매력적인 제목
2. 각 제품의 핵심 특징 강조
3. 카테고리별로 구분
4. 가격은 자연스럽게 표시
5. 행동 유도 문구 포함

결과를 product_announcement.txt로 저장.
```

**기대 결과:**
```
제목: 3월 신제품 출시 - 업무 효율을 높이는 프리미엄 IT 기기

안녕하세요, 고객님!

이번 달 출시된 신제품을 소개합니다.
```
