# 신호 프로토콜 (메모용)

라이브 중 숙번님이 지니에게 던지는 6가지 태그.

| 신호 | 의미 | 예시 |
|---|---|---|
| `[질문]` | 청중 질문 그대로 전달 | `[질문] Claude Code랑 Copilot 차이가 뭐예요?` |
| `[분위기]` | 청중 분위기 옮김 | `[분위기] 헷갈리는 표정이야` `[분위기] 빠져들었어` |
| `[요청]` | 페이스 조정 | `[요청] 천천히` `[요청] 더 깊이` `[요청] 예시 더` |
| `[시간]` | 시간 큐 | `[시간] 5분 남음` `[시간] 다음 beat 들어가자` |
| `[전환]` | 다음 beat로 | `[전환] Beat 4 시작` |
| `[직접]` | 청중에게 직접 / 마스터 키 | `[직접] 청중한테 인사 한번` `[직접] 진짜 의견?` |

## 마스터 키

**`[직접] 진짜 의견?`** — 지니가 sycophancy 모드로 빠질 때 한 마디로 끌어내는 핵심 신호.

## 설계 원칙

> "평소엔 자연스러운 대화, 예외 상황에만 명시적 신호"

Setlist는 재즈. 모든 cue를 형식화하면 즉흥감이 죽음.

## 시간 관리 책임 분담

- **숙번님**: 시간 추적·조정·상황 전달 전담
- **지니**: 작업 수행 + 막힘 시 즉시 노출
