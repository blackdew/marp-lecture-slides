# MCP — 공용 표준이 되다

- MCP는 AI와 도구를 잇는 '공용 콘센트' 규격 (2024년 등장)
- 월 다운로드가 1억 회 가까이로 폭증
- 기업 78%가 MCP 기반 에이전트를 실제로 운영
- 한 회사 것이 아니라 업계 공용 표준으로 관리됨

<div class="note">

표준이 생기면 도구가 늘고, 도구가 늘면 에이전트가 더 쓸모있어진다

</div>

출처 — digitalapplied, Wikipedia
