# Lecture 03: AI Agent 처음 만나기 - 강의 리뷰

## 📊 강의 개요

- **대상**: 시니어 (고령자)
- **주제**: AI 에이전트 기초
- **형식**: Marp 프레젠테이션 (16:9)
- **분량**: 391줄 (4강 통합 구성)

## ✅ 강점

### 1. 타겟 청중에 완벽히 최적화
- **용어 설명이 탁월**: "디지털 비서", "똑똑한 친구" 등 친근한 표현
- **단계별 난이도**: 1강(개념) → 2강(대화법) → 3강(활용) → 4강(안전)
- **실생활 예시 풍부**: 약 시간, 장보기, 음악 듣기 등

### 2. 세심한 배려
```markdown
> 버튼 위치를 모르겠다면 가족이나 강사에게 한 번 더 물어보세요.
> 버튼을 누르고 불이 켜지면 비서가 들을 준비가 됐다는 신호입니다.
```
- 기술에 익숙하지 않은 학습자를 위한 세밀한 안내
- 실패 시 대처법까지 포함 ("잘 안 들릴 때 이렇게 해보세요")

### 3. 반복적 강화 학습
- 각 강의 끝에 "정리해볼까요?" 섹션
- "다음 장 예고"로 연속성 유지
- 미션 체크리스트로 즉시 실습 유도

### 4. 안전 교육 포함
```markdown
## 1. 이것만은 꼭! 안전하게 사용하기

**주민등록번호, 은행 계좌번호, 비밀번호** 등
중요한 개인정보는 절대로 말하지 마세요.
```
- 4강 전체를 안전/보안에 할애
- 자주 묻는 질문 (Q&A) 섹션 포함

## ⚠️ 개선 가능한 점

### 1. 배경 이미지 크기 문제
**문제점:**
```markdown
![bg left:40%](images/img_p1_1.svg)
```
- CLAUDE.md 가이드에서는 35% 권장하지만 40% 사용
- 이미지가 슬라이드 가장자리에 붙을 위험

**개선안:**
```markdown
![bg left:35%](images/img_p1_1.svg)
```
모든 `![bg left:40%]`를 `![bg left:35%]`로 변경

### 2. 이미지 파일 일관성 부족
**문제점:**
- `img_p1_1.svg`, `img_p1_2.svg` ... 네이밍은 일관적
- **하지만** `img_p3_4.jpg` 같이 JPG가 섞여 있음
- SVG 권장 가이드와 불일치

**개선안:**
- 모든 이미지를 SVG로 통일
- 또는 JPG가 필요한 이유를 명시 (예: 사진이 필요한 경우)

### 3. 실습 미션이 너무 많음
**문제점:**
```markdown
## 연습 미션 체크리스트

- [ ] 내 스마트폰이나 AI 스피커에서 비서를 불러봤다.
- [ ] 시간이나 날씨처럼 쉬운 질문을 해봤다.
- [ ] 원하는 답을 듣지 못했을 때 다시 말해봤다.
- [ ] 오늘 배운 좋은 부탁 문장을 종이에 적어봤다.
```
- 4개 항목은 시니어에게 부담스러울 수 있음
- 강의 중 실시간으로 하기 어려움

**개선안:**
- 필수 미션 1-2개로 축소
- "집에 가서 해보세요" 미션 별도 구분
```markdown
## 지금 함께 해봐요 (필수)
- [ ] 비서 불러보기
- [ ] "지금 몇 시야?" 물어보기

## 집에 가서 해보세요 (선택)
- [ ] 다른 질문들 시도해보기
- [ ] 가족에게 보여주기
```

### 4. 기기별 차이 설명 부족
**문제점:**
- 아이폰, 갤럭시, AI 스피커를 모두 다루지만
- 각 기기의 **차이점**과 **어려움**에 대한 설명 부족
- 예: 빅스비는 설정이 꺼져있을 수 있음

**개선안:**
강사 노트 또는 별도 슬라이드 추가:
```markdown
## 강사 가이드: 기기별 주의사항

**아이폰 (Siri):**
- 홈 버튼 또는 측면 버튼
- "Hey Siri" 활성화 필요할 수 있음

**갤럭시 (Bixby):**
- ⚠️ 설정 > 고급 기능 > Bixby 활성화 먼저!
- 전원 버튼 길게 누르기

**AI 스피커:**
- 기기마다 깨우는 말 다름
- Wi-Fi 연결 필수
```

## 💡 우수 사례

### 1. 예시 시나리오 활용
```markdown
> 예시 시나리오: 김순자 어르신은 약 시간을 자주 잊었습니다.
> 비서에게 알람을 부탁한 뒤, 아침마다 "혈압약 드실 시간이에요"라는
> 안내를 받고 빠짐없이 챙겨 드셨습니다.
```
→ 실명(가명)과 구체적 상황으로 공감대 형성

### 2. 점진적 난이도 상승
1강: "AI 비서가 뭐예요?" (개념)
2강: "어떻게 말을 걸까요?" (기본 대화)
3강: "건강, 일상, 여가에 어떻게 쓰나요?" (활용)
4강: "안전하게 사용하려면?" (주의사항)

→ 완벽한 교육 설계

### 3. 안전 교육의 균형
- 너무 겁주지 않으면서도 필수 주의사항 전달
- "AI 비서가 가끔 틀릴 수 있다"는 현실적 설명
- 긍정적 톤 유지하면서 책임감 강조
