# 3부: 코딩의 첫걸음 내딛기

## 3.1 프로그래밍 언어란? (feat. 파이썬)

### 3.1.1 핵심 메시지

프로그래밍 언어는 컴퓨터와 소통하기 위한 '쉽고 명확한 약속(언어)'입니다.

### 3.1.2 전달 방식

* 비유: 외국인에게 길을 알려줄 때, 복잡한 문장 대신 "직진, 그리고 우회전"처럼 쉽고 명확한 단어를 사용하는 것과 같습니다.
* 파이썬 소개: 수많은 프로그래밍 언어 중에서도, 문법이 영어와 비슷해 사람이 읽고 이해하기 가장 쉬운 '파이썬'을 맛볼 것입니다.
* 첫인사: `print("Hello, World!")` 코드를 보여주고, `print()`는 화면에 괄호 안의 내용을 '보여줘(출력해줘)' 라는 뜻의 '약속어'라고 설명합니다.

### 3.1.3 슬라이드 아이디어

사람의 언어("컴퓨터야, 화면에 '안녕'이라고 보여줘")와 컴퓨터의 언어(`print("안녕")`)를 나란히 보여주며 비교하기.

## 3.2 코딩의 핵심 개념 3가지

### 3.2.1 핵심 메시지

세상의 모든 복잡한 프로그램도 결국 이 세 가지 개념의 조합으로 이루어집니다.

### 3.2.2 전달 방식 (비유 중심)

* 1. 변수 (Variable): '이름표를 붙인 상자'. 데이터를 잠시 저장해두고 나중에 다시 쓰기 위한 개념.
  * 비유: `내_나이 = 30` 이라고 하면, '내_나이' 라는 이름표가 붙은 상자에 30을 넣어두는 것.
* 2. 조건문 (Condition): '만약 ~라면, ~한다'. 특정 조건이 맞을 때만 정해진 행동을 하게 만드는 것.
  * 비유: "만약 비가 온다면, 우산을 챙긴다."
  * 코드 예시: `if 날씨 == "비": print("우산 챙기세요")`
* 3. 반복문 (Loop): '정해진 횟수만큼 ~을 반복한다'. 똑같은 일을 여러 번 시키고 싶을 때 사용.
  * 비유: "반 학생들 각자에게, 숙제를 제출하라고 말한다."
  * 코드 예시: `for 학생 in ["철수", "영희", "민준"]: print(학생 + ", 숙제 제출해!")`

### 3.2.3 슬라이드 아이디어

각 개념마다 이름표 상자(변수), 갈림길(조건문), 순환 화살표(반복문) 아이콘을 활용하여 시각적으로 표현.

## 3.3 간단한 데이터 다루기 실습 (눈으로 보는 코딩)

### 3.3.1 핵심 메시지

2부에서 본 '데이터'를 3부에서 배운 '코딩'으로 직접 다뤄보며, 어떻게 의미있는 정보를 얻는지 체험합니다. (청중이 직접 코딩하는 것이 아닌, 강사가 시연하며 원리를 설명)

### 3.3.2 전달 방식

* 상황 설정: "어느 작은 카페의 일주일간 커피 판매량 데이터가 있습니다. 하루 평균 몇 잔을 팔았을까요?"
* 데이터: `커피_판매량 = [15, 20, 18, 25, 30, 28, 22]`
* 논리 순서대로 설명:
  1. "먼저, 판매량 데이터를 `커피_판매량`이라는 **변수**(상자)에 저장합니다."
  2. "총합계를 저장할 `총_판매량`이라는 상자도 만들고, 일단 0을 넣어둡니다."
  3. "이제 **반복문**으로 `커피_판매량`에 있는 숫자를 하나씩 꺼내봅니다."
  4. "꺼낸 숫자를 `총_판매량` 상자에 계속 더해줍니다."
  5. "반복이 끝나면, `총_판매량`을 7로 나누어 평균을 구하고, 화면에 보여달라고 `print` 합니다."

### 3.3.3 슬라이드 아이디어

코드 전체를 한번에 보여주지 않고, 설명 순서에 따라 한 줄씩 나타나게 하며 각 줄이 어떤 의미인지 한글로 옆에 달아주기. 최종적으로 간단한 코드 몇 줄로 '평균 판매량'이라는 새로운 정보를 얻어내는 것을 보여주며 마무리.
